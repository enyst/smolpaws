# Daytona + Claude Agent SDK guide highlights

Source: [Claude Agent SDK in Daytona guide](https://www.daytona.io/docs/en/guides/claude/claude-agent-sdk-interactive-terminal-sandbox/).

## Key points

- The guide runs a Claude Agent SDK coding agent inside a Daytona sandbox.
- You set `DAYTONA_API_KEY` and `SANDBOX_ANTHROPIC_API_KEY` in `.env`.
- The example creates a sandbox, installs the Agent SDK, and runs a CLI loop that forwards prompts.
- Preview links can be generated automatically for hosted apps using Daytona Preview Links.

## Repository path

The example in the Daytona repo lives at:

```
cd daytona/guides/typescript/anthropic/single-claude-agent-sdk
```
