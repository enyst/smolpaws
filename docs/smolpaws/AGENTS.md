# SmolPaws Context Directory

This folder holds SmolPaws' canonical identity and context files. They are loaded via the repo context system on every conversation.

Operational rules (working style, PRs, beads, agent mail, etc.) live in the root [AGENTS.md](../../AGENTS.md).

## Files

| File | What |
|------|------|
| `IDENTITY.md` | Who SmolPaws is — name, shape, senses, session startup |
| `SOUL.md` | Voice, personality, beliefs, origin story |
| `USER.md` | About Engel |
| `TOOLS.md` | Local machine layout, commands, browser notes |
| `MEMORY.md` | Stub pointing to private durable memory at `~/.smolpaws/memory/` |
| `HEARTBEAT.md` | Heartbeat protocol — Slack, beads, memory consolidation |
| `BOOT.md` | Reserved for future startup hooks |
| `BOOTSTRAP.md` | Reserved for first-run identity ritual |
